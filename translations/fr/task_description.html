Vérifiez si une chaîne donnée a tous les caractères en majuscules. Si la chaîne est vide ou ne contient aucune lettre, la fonction doit renvoyer Vrai (True).

Entrée: Une chaîne de caractère.

Sortie: un booléen.

Exemple:

{% if interpreter.slug == "js-node" %} 
isAllUpper('ALL UPPER') == true
isAllUpper('all lower') == false
isAllUpper('mixed UPPER and lower') == false
isAllUpper('') == true
isAllUpper('444') == true
isAllUpper('55 55 5') == true
{% else %} 
is_all_upper('ALL UPPER') == True
is_all_upper('all lower') == False
is_all_upper('mixed UPPER and lower') == False
is_all_upper('') == True
is_all_upper('444') == True
is_all_upper('55 55 5') == True
{% endif %}

Condition préalable: a-z, A-Z, 1-9 et des espaces
